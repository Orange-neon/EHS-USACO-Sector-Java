# Problem 1: Array Sum

## Description
Given an array of integers, compute the sum of all elements.

## Input Format
- First line: integer N (number of elements)
- Second line: N space-separated integers

## Output Format
- Single line: "Sum: X" where X is the sum

## Example
**Input:**

5

1 2 3 4 5

**Output:**
Sum: 15

## Java Instructions
- Class must be named `Main`
- Use `Scanner` for input
- For testing, uncomment the `System.setIn` line
- Remove it for competition submissions

## How to Test
1. Open this folder in Codespaces
2. Compile: `javac Main.java`
3. Run: `java Main < test_cases/input_01.txt`
4. Compare output to `test_cases/output_01.txt`
